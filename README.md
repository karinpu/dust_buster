# DustBuster

**Цель:** автоматическая очистка «пыльных» (dust) токенов с вашего Ethereum‑кошелька.  
Скрипт ищет все ERC‑20 токены с эквивалентом менее порога (ETH) и готовит транзакции на их отправку на «сборный» адрес.
